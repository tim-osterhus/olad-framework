# Roadmap Checklist (Template)

Use this checklist to validate roadmap items before committing them to a milestone.

- [ ] Problem statement is clear and scoped
- [ ] Success metrics defined (what does “done” mean?)
- [ ] Dependencies identified (teams, services, vendors)
- [ ] Risks documented with mitigations
- [ ] QA/verification plan documented
- [ ] Rollout and rollback plan documented
- [ ] Documentation updates identified
- [ ] Owner assigned and timeline agreed
